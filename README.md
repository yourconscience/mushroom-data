# mushroom-data

Game data for [ГРИБ](https://yourconscience.github.io/mushroom/): zones, bonuses, leaderboard snapshots.

- `zones.js` — zone definitions (palette, enemies, speed, chiptune riff)
- `bonuses.js` — pickup definitions (jetpack, boost, loot)
- `leaderboard.json` — top-10 snapshot, refreshed by `dump_leaderboard.sh` from cron every 10 min
